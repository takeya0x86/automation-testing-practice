This site is a demo site for learning test automation.

This site is an ALPHA version.

This site is a Test object (System under test) used for learning browser test automation like Selenium.

It also can use for sample or demo on book and blog.

It is made for a learning test automation, but you can use a learning test design and technique.

Site Structures

This site is a mock hotel reservation. There are input forms for login, sign up and reserve. Responsive design is supported, so it can be displayed in mobile browsers.

Details

Login Page

This page contains simple text inputs and a button. The login information is stored in a Cookie. You can use the users you have saved on the sign up page, or preset one (see below).

Sign up Page

This page contains multiple types of inputs. The users registered on this page are stored in a Local Storage and can be used for login.

My Page

This page is displayed after you logged in. Registered user information is displayed and can be used for confirmation. And if the user is saved on the sign up page, you can set the icon image and cancel the membership (delete information).

Plans Page

This page is displayed from "Reserve" menu. The displayed plan will change depending on whether you are an "un-logged in", "normal member" or "premium member". Plan information is loaded asynchronously via Ajax (except for the top ones).

Reserve Page

This page is for making a reservation. Opens in a new window. This page contains multiple types of inputs, and the total price is dynamically calculated and displayed. The base rate and the number of nights and number of people allowed will change depending on the plan you selected.

Confirm Page

This page will be displayed after the reservation is confirmed. You will see a confirmation of the information you have entered in the reservation. And, an animated dialog will appear.
